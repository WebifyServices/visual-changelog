# Visual Changelog

Visual PR changelog plugin for Claude Code.

## Install

1. Add the marketplace in Claude Code:
   `/plugin` → Add marketplace → GitHub → `WebifyServices/visual-changelog`

2. Install the plugin:
   `/plugin` → Browse → visual-changelog → Install

## Usage

On any feature branch:
```
/visual-changelog:changelog
```

View the result:
```bash
./visual-changelog/run.sh
```

See [visual-changelog-plugin/README.md](./visual-changelog-plugin/README.md) for full documentation.
